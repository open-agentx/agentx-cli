# AGX Docs

This directory stores durable project knowledge that is not the live product source.

| Path | Purpose |
|---|---|
| `docs/adr/` | Long-lived decisions that should shape future work |
| `docs/runbooks/` | Repeatable operations, recovery steps, and agent workflows |
| `docs/sessions/` | Concise summaries of meaningful discussions |
| `docs/postmortems/` | Failure analysis and prevention notes |

Behavior contracts and non-trivial change plans live in `openspec/`.

Do not create new ad hoc root-level markdown files for durable project knowledge.
